# SALES-DASHBOARD
Project Title

Sales Performance Dashboard Analysis (Excel + Pivot + Charts + Slicers)

Project Summary 

Built an interactive sales dashboard using Microsoft Excel, Pivot Tables, Pivot Charts, and Slicers to analyze sales performance across months, sales channels, customer demographics, and order status. The dashboard enables quick business decision-making by tracking revenue trends, customer behavior, and operational performance.

Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting
Data Cleaning
Dashboard Design
Problem Statement

Businesses often have sales data but struggle to quickly identify:

Which months generate highest revenue?
Which sales channels perform best?
Who buys more—men or women?
Which age groups drive sales?
How many orders are successfully delivered?

This project solves these questions using an interactive dashboard.

Key Insights (very important)
1. Monthly Sales Trend
Sales peak in March
Revenue gradually decreases after mid-year
Q1 performs strongest
2. Sales Channel Performance
Amazon contributes highest share (~35%)
Myntra and Flipkart are strong secondary channels
Smaller channels contribute less than 10%
3. Order Status
Around 92% orders delivered
Very low cancellation/refund rate
Strong operational efficiency
4. Gender Analysis
Women contribute 64% of total sales
Men contribute 36%

This suggests female customers are the primary target audience.

5. Age Group Analysis
Adult segment generates highest sales
Women dominate across all age groups
Business Recommendations
Recommendation 1

Increase marketing budget on Amazon + Myntra

Recommendation 2

Launch women-focused campaigns

Recommendation 3

Target adult customer segment with personalized offers

Recommendation 4

Investigate sales decline in later months

Recommendation 5

Maintain delivery efficiency above 90%
